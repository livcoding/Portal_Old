STEPS TO BE FOLLOWED DURING ADVANCE LOAD DISTRIBUTION


Choose Faculty from the Top.
Click to choose Multiple/Additional Faculty and assign the proper SET from the link “Click to club Multiple/Additional Faculty globally” (if any).
Select (tick) the desired Batch to assign globally.
Use “Assign” Button to assign Main Faculty as well as Multiple/Addition Faculty globally.
Use Batch Merging here from the “Merge” drop down list only after clicking over the “Assign” Button.
Click to “Draft Save” Button to save after choosing everything you want to do.
Repeat above process in the same way to assign Load for different batches or to modify in the same batches in which Load assignment have been taken place earlier.
