Features:
---------
a) Table with multiple columns can be sorted very quickly.
b) Any type of data like numbers, string, date, currency, time can be sorted.
c) Easy to integrate into any website.
d) Simple and easily customizable JQuery sorter.

Downloads:
-----------
Please visit our site http://www.hscripts.com/scripts/jquery/table-sorter.php and do the download

Please take 5 minutes time and read installation instructions carefully and
completely!. 
Extracting files:
a) Unzip the table-sorter.zip file to extract the files.
b) You will get  table-sorter/index.php,table-sorter/jquery.js,table-sorter/README.txt,table-sorter/style.css and images.

Embedding the code:
a) Now Run the table-sorter/index.php in your browser.
b) JQuery Table Sorter script is now ready to work fast and efficient.
